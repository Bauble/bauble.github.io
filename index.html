Bauble

a web app for managing plant collections.

Modern

Bauble is built for the 21st century botanical garden. We leverage the latest web technologies so your plant records can grow with you.

Portable

Bauble works everywhere you go. Tablets, smartphones, desktops, you name it, we got it.

Open

Built completely with open-source you can be sure you plant records won't be locked up in a proprietary database.

Flexible

With an open and public API you can choose to use your data any way you would like. Feel free to integrate with third party platforms like Google Maps or Flickr.
